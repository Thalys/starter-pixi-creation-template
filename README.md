# Starter pixi Creation Template Project

Starter bootstrapped with `npm create pixi.js@latest`

## Pre-requisites

- bun
- git lfs

## Starting

```sh
# Copy the repo
tiged --mode=git git@github.com:Thalys/starter-pixi-creation-template.git shmup

# Initialize git
git init

# Ensure git lfs is installed first
git lfs install

# Install deps
bun i

# Run
bun dev
```
